# BejaClient Monorepo

Welcome to the **BejaClient Monorepo**, the official codebase for the high-performance BejaClient Minecraft launcher and client runtime. Fully optimized for competitive gameplay, maximum framerates, and seamless module management.

If you are a player looking to boost your in-game performance, you can download the latest release of the client **[here](https://github.com/bejameja/bejaclient/releases)**.

## Project Structure

This repository contains two primary packages. For detailed development, compilation, and contribution guides, please check their respective directories:

* **[Launcher Frontend (Electron)](#)** — The user interface, account manager, and launch settings.
* **[Client Runtime (Fabric Engine)](#)** — The custom game optimizations, title screen overrides, and performance injection mechanics.
